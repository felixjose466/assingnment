PROGRAMMING LANGUAGES

A programming lanuage is a formal language designed for communicate instructions to a machine, particularly a computer. Programming languages can be used to create programs that control the behavior of a machine and/ or to express algorithms precisely.

c++

C++(pronounced see plus plus) is a programming languages that is general purpose, statically typed, free-from, multi-paradigm and compiled. It is regarded as an intermediate-level language, as it comprises both high-level and low-level languages features

c

C is one of the most widely used programming languages of all time, and C comilers are available for the majority of available computer architectures and operating systems.

Java

Java is a computer programming language that is concurent, class-based, object-oriented, and specifically designed to have as few implementation dependencies as possible. It is intended to let application developers "write once, run anywhere"(WORA), meaning that code that runs on one platform does not need to be recompiled to run on another.
